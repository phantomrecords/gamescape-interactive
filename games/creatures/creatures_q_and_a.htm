Is there an Interactive Demo I can download?
No. It is not possible to produce an interactive demonstration of Creatures without giving the entire product away. Simply removing key features would affect usability seriously and the demo would not accurately represent the final product. A rolling demo is available as are some screen-shots on this web site.
Yes, but what about the one which was available in America?
This was an early pre-beta version of the title. It is no longer available as it no longer represents the product accurately.
Will there be a UNIX version? Or a LINUX one? Does Creatures run under NT? What about OS/2?
At present Creatures is available for Apple Macintosh and Windows '95 only. A Windows 3.1 version will be released within the next few months. No other versions are planned. Creatures does not function under any version of Windows NT. No fix for this is currently planned.
Whats this program all about? Is it like Dogz, Catz, or that game Little Computer People?
Creatures uses the proprietry CyberLife Artificial Life technology. To learn more, have a look at our What Is CyberLife page.
