System Requirements

  • Personal Computer with a Pentium Processor.
  • 8 megabytes of RAM to play in DOS, 16 megabytes to run in a Win95 DOS Box.
  • A Sound Blaster or 100% compatible sound card is highly recommended.
  • 40 megabytes available hard disk space.

Installation Tips

  • Unzip all Quake archive files into a temp dir.
  • Type INSTALL at the DOS prompt and follow instructions.
