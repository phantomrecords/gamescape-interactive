KKND Cheat Codes For Hex Editor

Open a save game file in a Hex editor, like Hex Workshop. Replace the first line (which won't look very different from this: 00000000 - 0100 0000 0000 0000 0000 0000 0000 0000) with this 00000000 - 0100 0000 0000 0000 FFFF FFFF FFFF 0000. This will give you lots of resources.
