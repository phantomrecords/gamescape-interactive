TALK TO GAMESCAPE FROM THE NET!

Welcome to Gamescape Customer Service:

Gamescape is the only Internet store to offer realtime communications with it's customers.

| Go Back | Next Page | This service supports Windows 95 only and uses Microsoft NetMeeting version 1.0 and above. You can use NetMeeting with any Internet browser as long as your connection the proper type (in other words a full Internet connection). If you are unsure just try it.

Using the all of the voice capabilities of NetMeeting require a microphone and soundcard with speakers connected. You can, however use NetMeeting to do traditional chat sessions with us (i.e. using the keyboard). Most people prefer this method to voice or because their sound is not setup with NetMeeting. To set up the sound use the NetMeeting Audio Tuning wizard. Or forget about sound and use your keyboard. We are quite happy to use our keyboard as well.

All the other features in NetMeeting work regardless of sound settings including: keyboard talking (chatting), file sharing, application sharing, and text-conferencing.
